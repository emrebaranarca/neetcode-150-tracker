# 🚀 NeetCode 150 Tracker

LeetCode mülakatlarına hazırlanmak için en etkili 150 soruyu takip edebileceğiniz interaktif bir çalışma planı.

![NeetCode 150 Tracker](https://img.shields.io/badge/Problems-150-00b341?style=for-the-badge) ![Blind 75](https://img.shields.io/badge/Blind%2075-Included-a78bfa?style=for-the-badge) ![Categories](https://img.shields.io/badge/Categories-18-ffa116?style=for-the-badge)

## ✨ Özellikler

- **150 Küratörlü Soru** — NeetCode 150 listesinin tamamı
- **18 Kategori** — Arrays, Trees, Graphs, DP ve daha fazlası
- **Blind 75 Filtresi** — En kritik soruları öne çıkar
- **İlerleme Takibi** — Çözdüğün soruları işaretle, tarayıcında kaydedilir
- **Zorluk & Kategori Filtreleme** — Easy, Medium, Hard veya kategoriye göre filtrele
- **Pattern Etiketleri** — Her sorunun hangi algoritmik pattern'i öğrettiğini gör
- **Arama** — Soru adı veya pattern'e göre ara
- **Mobil Uyumlu** — Telefon, tablet ve masaüstünde çalışır
- **Sıfır Bağımlılık** — Tek bir HTML dosyası, build gerektirmez

## 📖 Önerilen Çalışma Sırası

1. **Arrays & Hashing** ile başla (temel yapı taşları)
2. **Two Pointers → Sliding Window → Stack** (temel teknikler)
3. **Binary Search → Linked List** (veri yapıları)
4. **Trees → Heap → Tries** (ağaç yapıları)
5. **Graphs → Advanced Graphs** (graf algoritmaları)
6. **1-D DP → 2-D DP** (dinamik programlama)
7. **Greedy → Intervals → Math → Bit Manipulation** (tamamlayıcı konular)

Her kategoride **Easy → Medium → Hard** sırasını takip et. **B75** etiketli soruları öncelikli çöz.

## 🛠 Deploy

### Netlify
1. Bu repoyu fork edin
2. [app.netlify.com](https://app.netlify.com) → "Add new site" → "Import from Git"
3. GitHub reposunu seçin
4. Deploy edin — build ayarı gerekmez

### GitHub Pages
1. Repo Settings → Pages
2. Source: "Deploy from a branch" → `main` → `/ (root)`
3. Save
